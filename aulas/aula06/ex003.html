Principais tipos primitivos

Nota: existem outros que estão relacionados na aula 06

NUMBER to STRING - para converter um number para uma string usamos String(n) ou n.toString()

STRING to NUMBER - para converter uma string em número inteiro usamos o parserInt(n) para converter num número inteiro e parseFloat(n) para converter num número real, ou simplesmente usamos Number(n).

Nota: para fixar as casas decimais no parseFloat(n) usamos o .toFixed(dígitos) e nos dígitos inserimos quando dígitos de casas decimais teremos no número

Nota: TEMPLATE STRING OU MODELO DE STRING, podemos concatenar várias strings usando a template string que ficam entre crase e cada variável é retornada por um modelo ${variável dentro de colchetes} ou também o chamado PLACEHOLDER ${}.

Template String `string ${var} string ${var}` 

                
                    var nome = marcelo
                    var idade = 42
                    document.write(`meu nome é ${nome} tenho ${idade} anos`)
                    irá retornar: meu nome é marcelo tenho 42 anos
