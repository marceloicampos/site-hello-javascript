EVENTOS são qualquer interações que podem acontecer com qualquer elemento do HTML, o mais comum são eventos de mouse como mouseenter, mousemove, mousedown, mouseup, mouseout, click

PROCURE POR: javascript event list para a lista de eventos do javascript

FUNCTION é uma funcionalidade ou um conjunto de códigos que vão ser executadas quando um evento ocorrer, assim podemos usar as function para disparar eventos ou receber eventos que executam a funcionalidade.

UM BLOCO DE FUNCIONALIDADE É DELIMITADO POR SINAIS DE CHAVES

            
            function nameFunctionAreTheActionToDo(param, param) {
                BLOCO DE FUNCIONALIDADES
            }

LISTENER ou addEventListener são de formas disparar eventos através de "ouvidores" de interações, ou seja, os add Event Listener ficam ouvindo ou PRESTANDO ATENÇÃO no JavaScript para no momento em que acontecer o evento o mesmo dispare uma ação, funciona com qualquer elemento do DOM e não só para elementos do HTML
