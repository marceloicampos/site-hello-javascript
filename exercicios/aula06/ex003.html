Principais tipos primitivos

Nota: existem outros que estão relacionados na aula 06

NUMBER - para converter um number para string usamos String(n) ou n.toString()

STRING - para converter uma string em número inteiro usamos o parserInt(n) e o parseFloat(n) para converter num número real, ou simplesmente usamos Number(n).

Nota: TEMPLATE STRING OU MODELO DE STRING, podemos concatenar várias strings usando a template string que fica entre crase e cada variável é retornada por uma ${dentro de colchetes} ou também chamado PLACEHOLDER ${}.
Template String (`String ${} String ${}`) 

            
                var nome = marcelo
                var idade = 42
                document.write(`meu nome é ${nome} tenho ${idade} anos`)
                irá retornar: meu nome é marcelo tenho 42 anos

Boolean - que basicamente comporta valores lógicos como true or false.
