Trabalhando com Strings

ATRIBUTOS DE STRING
s.length diz quantos caracteres tem uma string
s.toUpperCase() colocar a string para caixa alta
s.toLowerCase() colocar a string em letras minúsculas

ATRIBUTOS DE NUMBERS
n.toFixed(n) diz quantas casas decimais tem um número
n.replace('.',',') troca o ponto pela vírgula
n.toLocaleString('pt-BR', {style: 'currency', currency: 'BRL'}) colocar o número com cifrão em Reais BRL
