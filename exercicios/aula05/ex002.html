Variáveis - É uma objeto em programação que serve basicamente servem para guardar dados

Nota: uma variável declarada usando a declaração var ou let sem especificar o valor inicial tem o valor undefined.

Nota: se uma variável é nula, o valor nulo se comporta como 0 em contextos numéricos e como falso em contextos booleanos.

var - serve para armazenar qualquer tipo de dado, os chamados tipos primitívos, em variáveis, que são desde os primários como 'Strings' ou cadeia de caracteres, 'Number', 'Infinity', 'NaN', 'Boolean True and False' e outros como 'null' que é um objeto para o JavaScript, 'undefined', 'object', 'array' ou vetor da programação e até tipos complexos como estruturas criadas pelo programador como uma 'function'.

Nota: pode ser usada para declarar tanto variáveis locais, dentro da função, como variáveis globais, fora da função.

Nota: usamos o command typeof para verificar a variável utilizada.

let - é muito semelhante ao var, no sentido de que podemos instanciar e armazenar qualquer tipo de objeto nelas, mas uma vez criadas as variáveis, quando utilizamos o let os seus valores podem ser sobrescritos, até mesmo por outros tipos de dados.

Nota: poder ser usada para declarar uma variável local de escopo do bloco. Nota: let significa do inglês letting or to allow or permit, ou seja, deixa passar ou permita.

const - funciona de forma análoga aos demais, entretanto, uma vez que o valor foi atribuído à variável, ele não pode ser modificado ou ser declarada novamente, ou seja, a variável é constante.

Nome da Variável ou Identificador da Variável

O nome da variável chamamos de identificador com as regras abaixo definidas

Para escrever nomes de variáveis:

  1. podemos começar com letras, $ ou _
  2. podemos usar letras e números
  3. podemos usar acentos e símbolos
  4. NÃO PODEM começar com números
  5. NÃO PODE conter espaços
  6. NAÕ PODE usar palavras reservadas como function ou alert ou var entres outras

NOTA: maiúsculas e minúsculas fazem diferença na hora de criar uma variável.

NOTA: é boa prática criar identificador de variáveis com nomes que sejam entendíveis para as próximas pessoas que forem analisar o código. Nomes coerentes para as variáveis coerentes.
